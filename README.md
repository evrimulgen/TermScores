# TermScores
Quickly and easily check NHL, NBA, and NFL scores from your command line.

# Requirements
    Python3 is required
    
    
# Usage
    
Download the source code, change to its directory, and run:
    
    pip install -r requirements.txt
    
    python3 termscores.py [league] [date]
